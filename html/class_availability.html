Detailed Description

Implementation of the Availability class methods.

The Availability class represents a dialog in the hotel management system for displaying the availability status of rooms. It is responsible for updating the user interface based on the current room availability information obtained from the Hotel class.

The class includes a constructor for initializing the dialog and setting its size. The readData method retrieves room availability information from the Hotel class, then adjusts the color of corresponding labels in the UI to indicate availability. The destructor ensures proper cleanup of UI resources.
