Detailed Description

Coordinates functionalities within the hotel management system.

The System class is responsible for handling reservations, services, customers, and other functionalities within the hotel management system. It provides methods for creating and canceling reservations, managing services, handling customer information, and more.

Note
Serving as a core component, this class ensures seamless interaction and coordination among different aspects of the system.
